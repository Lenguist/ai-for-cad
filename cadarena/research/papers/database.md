## Notes

1. **Deduplication:** Papers appearing across multiple source files (Round 1, Round 2, web search) have been merged into single entries. Preference was given to confirmed arxiv/doi links over placeholder URLs.
2. **Authors:** Some entries from the knowledge-base rounds had uncertain or incomplete author lists (listed as "Various"). These should be verified against actual publications.
3. **Links:** Entries with "N/A" links could not be confirmed via web search. Papers with confirmed arxiv or DOI links have been prioritized.
4. **Year assignment:** For papers with multiple versions (e.g., arXiv 2024 but conference 2025), the year reflects the publication/conference year where known.
5. **Categories:** Each paper is assigned exactly one primary category. Some papers span multiple areas (e.g., a generation paper that also introduces a dataset) but are categorized by their primary contribution.
