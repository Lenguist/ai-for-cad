Any LLM · methodology

Out of scope for text2cad benchmark
Methods excluded because they don't take text as primary input — cataloged here for completeness.
Category Examples Why excluded
Point cloud → CAD CAD-Recode, CAD-SIGNet, SECAD-Net, Point2Cyl, ComplexGen Input is 3D scan / point cloud, not text. Reverse engineering task.
Image → CAD CADCrafter, CADDreamer, CAD-Coder (image version), Scan2CAD Input is image / photo. Different benchmark track needed.
Unconditional generation DeepCAD, BrepGen, BrepGPT, AutoBrep, HoLa, BrepDiff, DTGBrepGen No text conditioning — generates random CAD shapes. No text prompt to benchmark against.
CAD editing (text + existing CAD) CAD-Editor, B-repLer, FlexCAD (conditional mode) Requires an existing CAD model as additional input. Different task.
Mesh-only output Meshy AI, Kaedim, Tripo (mesh mode) Outputs triangle mesh, not parametric CAD. Not editable / not engineering-grade.
Sketch / drawing → CAD Free2CAD, Sketch2CAD, PICASSO Input is hand-drawn sketch, not text.
