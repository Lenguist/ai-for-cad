  watertight           — no open edges (bool)

  vertex_count / face_count
  volume_mm3

  [WVS] = watertight + positive Volume + no Self-intersections
  lowercase = failed that check,  ? = unknown

--- Specific prompts ---

python score_vlm.py --method gpt-5.4 --prompts t1_01 t3_02

--- What it does ---
  1. Loads each .stl from outputs/
  2. Renders to 512x512 PNG (pyrender if installed, else matplotlib fallback)
  3. Sends image + original prompt to Claude Sonnet with a 0-10 rubric
  4. Scores: geometric correctness (0-5) + dimensional accuracy (0-5) = total (0-10)
  5. Writes vlm_score and vlm_score_detail into each <pid>.json

--- Cost estimate ---
  ~$0.01 per STL with Claude Sonnet (image input)
  20 prompts × 10 methods = 200 calls ≈ $2

  4. STEP 4 — HUMAN REVIEW
================================================================================

--- Open the review tool in your browser ---

open /Users/mbondarenko/Desktop/ai-for-cad/cadarena/eval/review.html

  Or just double-click review.html in Finder.

--- Setup in the browser ---
  1. Enter Method ID (e.g. "claude-opus-4-6")
  2. Load result files: select all *.json from methods/<id>/outputs/
     (exclude summary.json manually or it'll just be ignored)
  3. Load STL files: select all *.stl from methods/<id>/outputs/
  4. Optionally load existing human_review_<method>.json to resume
  5. Click "Start Review"

--- Keyboard shortcuts ---
  P        — mark PASS
  F        — mark FAIL
  Enter    — save + next prompt
  →        — save + next prompt
  ←        — previous prompt

--- When done ---
  Click "Export human_review.json" → downloads human_review_<method>.json

--- Merge back into the output JSONs ---

cd /Users/mbondarenko/Desktop/ai-for-cad/cadarena/eval
