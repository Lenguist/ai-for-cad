### Modal CadQuery executor
- Built and deployed `cadarena/modal_cadquery_executor.py`
- Uses `modal.Image.micromamba()` with `cadquery=2.4.0` + `apt_install("libgl1-mesa-glx", "libglu1-mesa")` for OCC deps
- Uses `@modal.fastapi_endpoint(method="POST")`
- Takes `{ code: str }`, returns `{ stlBase64: str | null, error: str | null, syntaxValid: bool }`
- **Live endpoint: `https://maksym-d-bondarenko--cad-arena-executor-execute.modal.run`**
- Smoke tested: returns valid STL base64 for a simple box

### Wired Modal into /api/generate
- Added `executeCadQuery()` in `cadarena/site/app/api/generate/route.ts`
- Reads `MODAL_EXECUTOR_URL` env var (graceful degradation if unset — shows code only)
- Called for all `outputType === "cadquery"` models after code generation

### Upstash rate limiting
- Installed `@upstash/ratelimit` and `@upstash/redis`
- Created `cadarena/site/middleware.ts` — Vercel Edge middleware
- 5 requests/day per IP, 24h sliding window, applies only to `/api/generate`
- Graceful degradation: if `UPSTASH_REDIS_REST_URL`/`TOKEN` not set, all requests pass through
- Returns 429 with clear error message when exceeded
